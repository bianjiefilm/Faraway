## English

### What is Faraway?

Faraway is a menu bar app for macOS that helps you protect your eyes using the 20-20-20 rule: every 20 minutes, look at something 20 feet away for 20 seconds.

### Features

- 🌻 **Smart Reminders** - Automatically reminds you to take eye breaks
- 🎯 **App Monitoring** - Monitors when you're using editing apps (video, image, code editors)
- ⏱️ **Customizable Timer** - Set your preferred break interval
- 🎨 **Beautiful Design** - Sunflower-themed UI with illustrations
- 🚀 **Launch at Login** - Option to start automatically on Mac boot
- 📊 **Daily Summary** - Track your eye care progress

### Installation

1. Download the latest DMG from [Releases](https://github.com/yourusername/Faraway/releases)
2. Open the DMG file
3. Drag Faraway.app to your Applications folder
4. Launch Faraway from Applications

### First Launch

On first launch, you'll be asked if you want to enable "Launch at Login" for automatic starting.

### Usage

1. **Menu Bar Icon**: Look for the sunflower eye icon in your menu bar
2. **Click to Open**: Click the icon to open the popover
3. **Start Monitoring**: The timer starts automatically when you use editing apps
4. **Take Breaks**: When the timer ends, a beautiful reminder will appear
5. **Settings**: Customize your experience in the settings section

### Settings

- **Monitoring Mode**:
  - Global Mode: Monitor all apps
  - Select Apps: Choose specific apps to monitor
- **Launch at Login**: Enable/disable automatic startup

### System Requirements

- macOS 13.0 (Ventura) or later

## 中文

### 什么是 Faraway？

Faraway 是一款 macOS 菜单栏应用，采用 20-20-20 护眼法则：每 20 分钟，看一眼 20 英尺（约 6 米）远的地方，持续 20 秒。

### 功能特点

- 🌻 **智能提醒** - 自动提醒您休息眼睛
- 🎯 **应用监测** - 监测您何时在使用编辑类应用（视频、图片、代码编辑器）
- ⏱️ **自定义计时器** - 设置您偏好的休息间隔
- 🎨 **精美设计** - 向日葵主题 UI，搭配精美插画
- 🚀 **开机启动** - 支持开机自动启动
- 📊 **每日总结** - 追踪您的护眼进度

### 安装方法

1. 从 [Releases](https://github.com/yourusername/Faraway/releases) 下载最新的 DMG 文件
2. 打开 DMG 文件
3. 将 Faraway.app 拖入应用程序文件夹
4. 从应用程序中启动 Faraway

### 首次启动

首次启动时，会询问是否开启"开机启动"选项。

### 使用方法

1. **菜单栏图标**: 在菜单栏找到向日葵眼睛图标
2. **点击打开**: 点击图标打开弹出窗口
3. **开始监测**: 当您使用编辑类应用时，计时器会自动开始
4. **休息提醒**: 计时结束时会弹出精美的休息提醒
5. **设置**: 在设置区域自定义您的体验

### 设置说明

- **监测模式**:
  - 全局模式: 监测所有应用
  - 手动选择: 选择特定应用进行监测
- **开机启动**: 开启/关闭开机自动启动

### 系统要求

- macOS 13.0 (Ventura) 或更高版本
